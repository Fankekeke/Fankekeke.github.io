SpringData_JPA

Spring Data JAP:
Spring Data JAP是 spring data 项目下的一个模块。提供了一套基于 JPA标准操作数据库的简化方案。底层默认的是依赖 Hibernate JPA 来实现的。

Spring Data JPA 的技术特点：
我们只需要定义接口并集成 Spring Data JPA 中所提供的接口就可以了。不需要编写接口实现类

Repository 提供了两种查询方式的支持
1）基于方法名称命名规则查询
