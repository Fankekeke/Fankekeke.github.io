@Cacheable 与@CacheEvict

@Cacheable

@Cacheable 作用：把方法的返回值添加到 Ehcache 中做缓存

Value 属性：指定一个 Ehcache 配置文件中的缓存策略，如果么有给定 value，name 则表示使用默认的缓存策略。

Key 属性：给存储的值起个名称。在查询时如果有名称相同的，那么则知己从缓存中将数据返回

@CacheEvict

@CacheEvict 作用：清除缓存
